Nginx Tutorial Nginx is an open source, fast, lightweight and high-performance web server that can be used to serve static files. NGINX is considered as the popular web server behind Apache web server and Microsoft's IIS. NGINX improves content and application delivery, improves security, and facilitates scalability and availability for the busiest websites on the internet.
