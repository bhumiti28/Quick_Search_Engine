In our example, we have explained the simple way to get the difference. We will use date and math to get the number of days between dates. Math() and Date() functions are the JavaScript function, and we are using them to get the difference between two dates in days. That's why we cannot say that we are getting the number of days between dates by using jQuery. By using math and the date function, we cannot only get the difference between dates in days, but we can also get it in months and years also. In the below example, we will perform the task by creating a daydifference() function, which will contain two arguments, i.e., first date and last date. The difference in days will be returned by this function. In our jQuery application, we have to simply copy that code to get the difference between dates in days. Using the following example, we are able to get the difference. We can also use this code in our jQuery code. Example: 
<!DOCTYPE html>
<html>
<head>
    <title> Jquery example to get number of days between two dates </title>
</head>
<body>
  
<script type="text/javascript">
   
var days = daysdifference('03/19/2021', '03/31/2021');
// Add two dates to two variables  
console.log(days);
  
function daysdifference(firstDate, secondDate){
    var startDay = new Date(firstDate);
    var endDay = new Date(secondDate);

// Determine the time difference between two dates   
    var millisBetween = startDay.getTime() - endDay.getTime();

// Determine the number of days between two dates
    var days = millisBetween / (1000 * 3600 * 24);

// Show the final number of days between dates   
    return Math.round(Math.abs(days));
}
</script>
   
</body>
</html>
 The approach of the above example is as follows: Firstly, we will define two dates. Then we will use startDay.getTime()- endDay.getTime() to determine the time difference between two dates. After that, we will determine the number of days between dates. We will do this by dividing the number of milliseconds in a day and the time difference of both dates.
